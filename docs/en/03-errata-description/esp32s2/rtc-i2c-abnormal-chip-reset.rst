[I2C-97] The Falling Edge of RTC_I2C_RESET Triggers Reset at Low Temperature
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32s2

   .. tags::

      v0.0

Description
^^^^^^^^^^^

At –40 °C, the chip will be restarted during wake-up.

Workarounds
^^^^^^^^^^^

None.

Solution
^^^^^^^^

Fixed in chip revision :bdg-success:`v1.0`.
