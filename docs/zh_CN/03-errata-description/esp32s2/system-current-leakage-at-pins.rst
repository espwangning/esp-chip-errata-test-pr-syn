[系统] VDDA 和 VDD3P3_RTC 管脚在芯片关闭时漏电流
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32s2

   .. tags::

      v0.0

描述
^^^^

当芯片电源供电，且 CHIP_PU 管脚拉低、芯片处于关机状态下时，某些芯片的电源管脚 VDDA 和 VDD3P3_RTC 上会有 μA 级别的漏电。

变通方法
^^^^^^^^

无。

解决方案
^^^^^^^^

已在芯片版本 :bdg-success:`v1.0` 中修复。
