[GPIO] 当一些 RTC 外设的电源打开时，GPIO36 和 GPIO39 的数字输入会被拉低约 80 ns
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32

   .. tags::

      v0.0, v1.0, v1.1, v3.0, v3.1

描述
^^^^^^^^

打开以下 RTC 外设的电源会发生此问题：

- SAR ADC1 传感器
- SAR ADC2 传感器
- AMP 传感器

变通方法
^^^^^^^^

当用户决定把用于控制以上传感器的电源域打开时，应当忽略来自 GPIO36 和 GPIO39 的输入。
