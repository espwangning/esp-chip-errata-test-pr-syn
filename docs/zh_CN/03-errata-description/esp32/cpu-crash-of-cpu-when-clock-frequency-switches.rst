[CPU] CPU 频率切换导致系统卡顿
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32

   .. tags::

      v0.0

描述
^^^^^^^

CPU 频率从 240 MHz 直接切换到 80/160 MHz 会导致系统卡顿

变通方法
^^^^^^^^

建议使用以下两种模式：

    1. 2 MHz <-> 40 MHz <-> 80 MHz <-> 160 MHz
    2. 2 MHz <->40 MHz <->240 MHz

解决方案
^^^^^^^^^^^

已在芯片版本 :bdg-success:`v1.0` 中修复。
