[TWAI 控制器] 总线关闭恢复后发送的数据出错
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32

   .. tags::

      v0.0, v1.0, v1.1, v3.0, v3.1

描述
^^^^^^^^

总线关闭恢复完成后，TWAI 控制器下一次发送的数据可能出错（即不符合 TWAI 数据帧格式）。

变通方法
^^^^^^^^

一旦通过错误报警限制中断检测到总线关闭恢复完成，TWAI 控制器应先进入复位模式来复位控制器的内部信号，随后退出复位模式。
