[WIFI-9696] {IDF_TARGET_NAME} 无法作为 802.11mc 精确时间测量 (FTM) 的发起方
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32c6

   .. tags::
      
      v0.0, v0.1

描述
^^^^

802.11mc FTM 中 T3 的时间（即发起方发送 ACK 的时间）无法正确获取，因此无法作为 FTM 的发起方。

变通方法
^^^^^^^^

无变通方法。

解决方案
^^^^^^^^

已在芯片版本 :bdg-success:`v0.2` 中修复。
