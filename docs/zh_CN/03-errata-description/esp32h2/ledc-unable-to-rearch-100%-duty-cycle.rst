[LEDC-253] 分辨率设置成最大时，LEDC 无法达到 100% 占空比
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32h2

   .. tags::
      
      v0.0, v0.1

描述
^^^^

定时器选择使用最大占空分辨率时，LEDC 无法达到 100% 占空比。强制将占空比设置成 2\ :sup:`MAX_DUTY_RES` 将会影响内部占空校准。

变通方法
^^^^^^^^^

无变通方法。

解决方法
^^^^^^^^

已在芯片版本 :bdg-success:`v1.2` 中修复。
