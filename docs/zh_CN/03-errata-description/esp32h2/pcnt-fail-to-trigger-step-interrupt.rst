[PCNT] 无法触发步进中断
~~~~~~~~~~~~~~~~~~~~~~~

.. only:: esp32h2

   .. tags::
      
      v0.0, v0.1, v1.2

描述
^^^^

如果启用了步进计数器 (step counter) 且计数达到配置的低限值或高限值时，步进中断未按预期触发。

变通方法
^^^^^^^^

配置系统在计数器每增加或减少特定值 (x) 时通知应用程序。当满足此条件时，程序进入中断，读取当前计数器值，并调用用户的回调函数，将计数器值传递给调用方。

解决方案
^^^^^^^^
